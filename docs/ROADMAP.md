# Roadmap

本文件整理自原始待办记录，并结合当前代码结构归纳为后续优化方向。

## 待优化

- 修复导出完整图纸底部色号总览中中文“颗”字在部分环境下显示乱码的问题。
- 阶段 4 的缩放控件继续增强文字输入体验，并保持与快捷缩放行为一致。
- 阶段 4 支持从图纸上吸取颜色，用吸取到的色号作为当前编辑颜色。
- 上游图像处理阶段可探索接入 AI 绘画或风格化能力，用于生成更适合拼豆化的美术图。

## 发布前建议

- 增加几张不同尺寸、不同背景复杂度的示例输入和输出截图。
- 补充许可证文件。
- 在 GitHub Releases 中发布 Windows 压缩包，源码仓库只保留必要源码和示例。
